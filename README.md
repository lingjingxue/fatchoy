# fatchoy

Gung Hay Fat Choy(恭喜發財)


A set of libraries to create online game server.

`go get -u -v qchen.fun/fatchoy`

## 各目录说明

  目录       |  描述
------------|------------
codec       | 编解码
codes       | 错误码
debug       | 调试API
discovery   | 服务发现
packet      | 消息结构
l0g         | 日志
qnet        | 网络通信
sched       | 执行器
x           | 工具包
